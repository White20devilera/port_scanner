# Network Port Scanner
A high-speed asynchronous port scanning tool designed for network reconnaissance. It identifies open TCP ports on a target IP address to map out available services and potential entry points.



# Key Features:

Multi-threaded scanning for rapid results.

User-defined target IP and port ranges.

Detailed reporting of open ports and associated services.
